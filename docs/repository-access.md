# Repository Access

Canonical repository for this project:

- HTTPS: `https://github.com/imarinmed/sagg.git`
- SSH: `git@github.com:imarinmed/sagg.git`

Useful commands:

- Verify remotes: `git remote -v`
- Set origin if missing: `git remote add origin https://github.com/imarinmed/sagg.git`
- Update existing origin: `git remote set-url origin https://github.com/imarinmed/sagg.git`
- Push current branch: `git push -u origin <branch>`

Troubleshooting:

- If push fails with "No such remote 'origin'", add origin with the command above, then push again.
